• 사용대상 - 유동 IP를 사용하는 일반 인터넷 사용자 (권장수준)
    보안효과 - 로그인 후 IP주소가 최근 로그인 시 IP주소들과 일정범위(C클래스)
    까지 동일한 경우만 허용, 멀리 떨어진 장소에서
    로그인 권한을 부정사용 하는 것을 차단
  • 사용대상 - 유동 IP의 일반 인터넷에서 높은 수준의 IP보안을 원하는 사용자
    보안효과 - 로그인 후 IP주소가 최근 로그인 시 IP주소들과 동일한 경우만
    허용    , 인터넷 접속중 IP주소 변경 시 사용자 재확인을 최소화하고,
    3단계와 동일한 보안수준을 제공
  • 사용대상 - 인터넷 접속 중 IP주소가 변경되지 않는 사용자
    보안효과 - 로그인 후 IP주소가 변경되지 않는 경우만 허용, IP주소정보를
    사용한 최고의 보안수준을 제공
  • 로그인 상태에서 IP주소가 자주 변경되는 사용자의 경우 선택.
    단, IP보안 설정을 하지 않은 상태이므로 보안에 취약.

* 로그인 상태유지 설정 시 IP보안은 '사용안함'으로 변경되는
